  <action>
    Create `src/lib/report-stats.ts` with three pure exported functions and two exported interfaces per D-01/D-02:

    1. **MoodDistributionItem** interface: `{ emotion: string; emoji: string; count: number; percentage: number }`

    2. **StreakResult** interface: `{ current: number; best: number }`

    3. **computeMoodDistribution(entries)**: Accept array of `{primary_emotion: string, emoji: string}`. Group by `primary_emotion`, count occurrences, compute percentage as `Math.round((count / total) * 100)`, sort descending by count. Use the emoji from the first entry seen for each emotion. Return empty array if no entries.

    4. **computeDaysJournaled(entries)**: Accept array of `{created_at: string}`. Extract unique day strings using `new Date(e.created_at).toISOString().slice(0, 10)`, return the Set size.

    5. **computeStreak(entries)**: Accept array of `{created_at: string}`. Extract unique day strings, sort ascending. Compute best streak by iterating sorted days and counting consecutive runs (diff === 1 day). Compute current streak by counting backwards from today (use `new Date().toISOString().slice(0, 10)` for today, start from yesterday if today has no entry). Return `{current, best}`.

    Create `__tests__/lib/report-stats.test.ts` following the pattern from `__tests__/lib/mood-themes.test.ts`. Test all three functions with edge cases: empty input, single entry, multiple same-day entries, consecutive days, non-consecutive days, current streak ending today vs yesterday.
